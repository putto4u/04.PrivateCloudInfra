## 클러스터 외부 MySQL 서버 설치하기 
but 워커노드 VM01 에서 파드가 아닌 APT 설치

1. APT로 Mysql 서버 설치하기
   ```
   # 패키지 인덱스 업데이트
     sudo apt update

   # MySQL 서버 설치
     sudo apt install mysql-server -y
   ```
   
3. 접속 확인하기
   ```
   sudo systemctl status mysql  # active 확인하기
   sudo mysql -u root -p
   ```
   
5. 외부 접속 허용 설정
   ```
   sudo vi /etc/mysql/mysql.conf.d/mysqld.cnf
      bind-address = 127.0.0.1 부분을 0.0.0.0으로 수정하거나 주석 처리

   sudo systemctl restart mysql   # 설정 적용 위해 재시작
   ```
