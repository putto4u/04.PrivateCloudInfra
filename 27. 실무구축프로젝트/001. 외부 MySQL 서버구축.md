## 클러스터 외부 MySQL 서버 설치하기 
but 워커노드 VM01 에서 파드가 아닌 APT 설치

1. APT로 Mysql 서버 설치하기
   ```
   # 패키지 인덱스 업데이트
     sudo apt update

   # MySQL 서버 설치
     sudo apt install mysql-server -y
   ```
   
3. 접속 확인하기 및 사용자/비번 설정
   ```
   sudo systemctl status mysql  # active 확인하기
   sudo mysql -u root -p
   -- 데이터베이스 생성
   CREATE DATABASE my_database;

   -- 사용자 생성 및 비밀번호 설정
   CREATE USER 'my_user'@'%' IDENTIFIED BY 'StrongPassword123!';

   -- 권한 부여
   GRANT ALL PRIVILEGES ON my_database.* TO 'my_user'@'%';

   -- 변경 사항 적용
   FLUSH PRIVILEGES;
   EXIT;
   ```
